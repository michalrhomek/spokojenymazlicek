USAGE

Follow this steps to transfer products from SOURCE Prestashop website to TARGET Prestashop website.

TARGET website

  • find the ID number of the appropriate tax rate group
  • import the images from the SOURCE website product image folder (img/p) into a folder of the TARGET website. Do not use img/p at the target website

SOURCE website

  • Install the module
  • Save settings, the tax rate and image directory are of the TARGET website
  • Create the categories and product csv files (usually several numbered product files are created)
  • Optional: download the csv files and edit them in a spreadsheet editor if required

TARGET website

  • Upload the csv files into the admin/import folder
  • In the backoffice, open the Tools - Import tab
  • Import categories, skip the first line if used for description. Check "delete existing categories" if appropriate
  • Import the first numbered product file, skip the first line if used for description. Check "delete existing products" if appropriate
  • Import remaining product files, DO NOT DELETE existing products
  • If the images are not visible, open the settings / images tab , edit available image types and assign them to product as appropriate
  • Regenerate the product miniatures. It is recommended to regenerate one type of image at time
  • If the categories are not displayed at the frontend, uninstall and install the categories module
